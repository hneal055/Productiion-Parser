API Documentation

Complete API documentation for integrating with AURA Dashboard.

Authentication

POST /api/v1/auth/login

Authenticate and receive API token

Headers: Content-Type: application/json

Body: {"username": "string", "password": "string"}

Metrics Endpoints

GET /api/v1/metrics/system

Retrieve system metrics

Headers: Authorization: Bearer {token}

GET /api/v1/metrics/cpu

Retrieve CPU usage data

Parameters: range (optional) - time range for data

GET /api/v1/metrics/memory

Retrieve memory usage data

Alert Endpoints

GET /api/v1/alerts

Retrieve active alerts

POST /api/v1/alerts

Create a new alert
