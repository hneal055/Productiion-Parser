HOW IT WORKS:
───────────────────────────────────────────────────────────────────────────────
1. User uploads Budget 1 → Saved to analysis_cache + budget_history
2. User uploads Budget 2 → Saved to analysis_cache + budget_history
3. User clicks "Compare Budgets" → Shows history page
4. User selects 2 budgets → Submits form
5. compare_budgets() runs → Analyzes differences
6. Results stored in cache → Redirects to results page
7. Charts + tables generated → Displays comparison
8. User can print/export → Professional report ready
