Developer API

ScreenFlow AURA REST API

Integrate screenplay intelligence directly into your pipeline. A production-grade REST API with API key authentication, per-key rate limiting, usage metrics, and full JSON responses — ready for any language or platform.

Endpoint Method Auth Description Rate Limit
/api/health GET None Service status & analysis count —
/api/parse POST API Key Full screenplay parse — metrics, genre, quality score, themes, recommendations 20 / hr
/api/analyze POST API Key Deep narrative analysis — character depth, commercial viability, risk, comps 20 / hr
/api/validate POST API Key WGA/Hollywood format compliance check — issues, severity, certification status 20 / hr
/api/batch/parse POST API Key Batch process up to 10 scripts — parallel scoring with quick-verdict summaries 5 / hr
/api/history GET API Key Paginated analysis history with full result JSON —
/api/metrics GET API Key Usage metrics broken down by endpoint type —
/api/admin/keys POST Admin Token Create new API key — raw key returned once, HMAC-hashed at rest —
