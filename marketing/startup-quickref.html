Troubleshooting
# Port already in use? Find and kill the process: netstat -ano | findstr :PORT taskkill /PID <PID> /F # Missing module error? Rebuild venv: cd <project-folder> python -m venv venv venv\Scripts\pip install -r requirements.txt # API returning 500? Demo mode must be ON in .env: DEMO_MODE=true <— required (no live API credits)
