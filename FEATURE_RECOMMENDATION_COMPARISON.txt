ALGORITHM LOGIC:
───────────────────────────────────────────────────────────────────────────────

1. Normalize both DataFrames (same columns, cleaned data)
2. Calculate totals and high-level metrics
3. Group by Department, Category, Vendor
4. Compare each group:
   - If in both: Calculate difference

   - If only in budget1: Mark as "removed"
5. Sort by magnitude of change
6. Generate insights based on patterns
7. Create visualizations
