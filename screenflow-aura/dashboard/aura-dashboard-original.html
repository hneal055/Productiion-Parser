API Documentation

Overview

The AURA Dashboard API provides programmatic access to monitoring data and system controls. All API requests should be made to the base URL:

https://api.auradashboard.com/v1

Authentication

Include your API key in the request header:

Authorization: Bearer YOUR_API_KEY

Endpoints

Get System Metrics

Retrieve current system performance metrics.

GET /metrics

Get System Status

Check the overall system status and health.

GET /status

Create Alert

Create a new system alert.

POST /alerts
Content-Type: application/json

{
  "type": "warning",
  "message": "High CPU usage detected",
  "severity": "medium"
}

Retrieve Logs

Get system logs with optional filtering.

GET /logs?limit=50&type=error
