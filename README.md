# Production Budget Parser

A tool for parsing and analyzing production budgets for film and TV, with focus on risk management.

## Quick Start

1. Install requirements:
pip install -r requirements.txt
python src/main.py --input path/to/budget.xlsx --output ./data/output
- `src/` - Source code
- `data/input/` - Place input budget files here
- `data/output/` - Generated reports will be saved here
